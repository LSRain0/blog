## 为什么要写博客？

写博客是一种很好的学习方式。通过写作，我可以：

1. **整理思路**：把零散的知识点串联起来
2. **加深理解**：教是最好的学
3. **记录成长**：回头看自己的进步
4. **分享交流**：和志同道合的人交流

## 一段示例代码

```python
def fibonacci(n: int) -> int:
    """计算斐波那契数列的第 n 项"""
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b

# 测试
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
